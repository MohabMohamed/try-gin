# try-gin
A todo website just for testing golang web framework gin

## installation

### sqlite

Debian based(Ubuntu, Debian etc.) :

` 
$ sudo apt-get update 

$ sudo apt-get install sqlite3
` 
RPM based(RHEL, CentOS, Fedora etc.) :

` 
$ sudo yum update 

$ sudo yum install sqlite
` 

### try-gin

` 
$ go get 
`
## Run
` 
$ make dev
`
## Testing
` 
$ make test
`
